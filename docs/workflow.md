# Workflow

## 1. High-Volume Content Production (The Content Factory)

The Brainrot Report 666 utilizes a "Content Factory" model designed to handle daily or multi-daily postings without burnout.

### The Pipeline

Every piece of content moves through these directories:
1. **`content/drafts/`**: Raw scripts, ideas, and outlines.
2. **`content/scheduled/`**: Approved content that has been assigned a publishing date.
3. **`content/published/`**: Live content.
4. **`content/archive/`**: Old content stored for reference.

This pipeline ensures that we can batch-create content on a single day (e.g., Sunday) and let the automated system (via our dashboard and scripts) manage the release schedule throughout the week.
